# Portfolio UI kit

Pixel-recreation of the kakde.eu homepage as JSX components. Single-page scroll: Hero → About → Experience → Projects → Certifications → Cortex preview → Footer.

## Files
- `index.html` — interactive demo (sticky nav, dark-mode toggle, tab switching)
- `Header.jsx` — sticky top bar, pill links, theme toggle
- `Hero.jsx` — outlined card, two CTAs
- `About.jsx` — copy + 4-portrait collage with blue tint
- `Experience.jsx` — vertical company tabs + results panel
- `Projects.jsx` — 3-up grid of project cards
- `Certifications.jsx` — vertical timeline with star/trophy markers
- `CortexTeaser.jsx` — eyebrow + 4 chapter cards + browse-all
- `Footer.jsx` — Made with ♥ + brand icons
- `BrandIcons.jsx` — GitHub / LinkedIn paths
- `data.js` — projects, certifications, experiences (subset)
