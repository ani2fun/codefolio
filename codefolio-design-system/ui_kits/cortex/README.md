# Cortex UI kit

Pixel-recreation of the Cortex reader — three-column layout (sidebar / main / right-rail TOC minimap). Renders Markdown with syntax-highlighted code blocks.

## Files
- `index.html` — interactive demo with chapter switching and TOC scrolling
- `CortexSidebar.jsx` — book + chapter tree
- `CortexBreadcrumb.jsx` — Home / Book / Chapter trail
- `ChapterContent.jsx` — heading, prose, code blocks, callouts
- `CortexToc.jsx` — right-rail minimap (ticks → full TOC on hover)
- `CortexPager.jsx` — Previous / Next chapter strip
- `data.js` — sample book + chapters (subset)
